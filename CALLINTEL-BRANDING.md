# Callintel Product Branding

## Overview

"Callintel" has been added as the primary product name throughout the website, with "by JetBuilder" as the company attribution and "Tactical AI" as the tagline.

## Brand Hierarchy

```
Callintel              ← Product Name (Primary, Large, Tactical Orange)
   ↓
by JetBuilder          ← Company Name (Secondary, Small, Gray)
   ↓
Tactical AI            ← Product Category/Tagline (Tertiary, Small, Gray)
```

## Visual Identity

**Logo Structure:**
```
      JetBuilder        (small, gray, uppercase)
     CALLINTEL         (large, tactical orange, bold, uppercase)
     Tactical AI        (small, gray, uppercase)
```

**Color Scheme:**
- Product Name (CALLINTEL): `text-tactical-400` (tactical orange)
- Company Name (JetBuilder): `text-gray-400` or `text-gray-500`
- Tagline (Tactical AI): `text-gray-500`

## Logo Usage Guidelines

### Primary Logo (Header/Footer)
```
Three-line stacked format:
Line 1: JetBuilder (small, 12px, gray, uppercase)
Line 2: CALLINTEL (24px, orange, bold, uppercase)
Line 3: Tactical AI (12px, gray, uppercase)
```

### Chatbot Header
```
Two-line format:
Line 1: Callintel Assistant (bold, white, uppercase)
Line 2: Powered by JetBuilder (small, gray)
```

### Icon/Avatar
```
Single letter: "C"
Background: Tactical orange (#D97706)
Text: White
```
